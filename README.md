- **Platform:** Visual Studio Code / CodeBlocks

## Implemented Algorithms

### Greedy Algorithms
1. Activity Selection Problem
   - Selects the maximum number of non-overlapping activities.
   - Strategy: Sort activities by finishing time and greedily choose compatible activities.

2. Fractional Knapsack Problem
   - Maximizes total profit within a given knapsack capacity.
   - Strategy: Sort items by profit-to-weight ratio and take items greedily.


## How to Compile and Run

Using G++:

```bash

```

Example:

